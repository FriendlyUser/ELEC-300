Pulse Width Modulation System
The simple embedded system can be subdivided into independent modules:

  • Using a STM Microcontroller, standard electric equipment (multi-meters and oscilloscope), my partner and I created a system that displays updated frequencies and resistances when the potentiometer is changed.
  • TIM2 (unit) — given an analog signal, compute and display the frequency on the Eclipse debugger
  • ADC (unit) — take an analog signal and convert it to a digital one, so the STM Microcontroller can process it and compute the frequency
  • DAC (unit) — take the converted digital signal and output the analog signal again
  • SPI — Library that Interfaces with a LCD device to display the resistance and frequency
